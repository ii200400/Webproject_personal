# 웹 프로젝트 (필라테스 홈페이지)

재능기부의 일환으로 사용가능한 웹 페이지를 구현해보도록 한다.

## 개요

+ 제작 인원 : 3명
+ 개발 기간 : 약 3개월 (2017.09~2017.12)

## 개발 환경

+ Atom (텍스트 편집기)
+ WAMP (오픈 소스 통합 소프트웨어 번들)

## 사용 기술

+ Front-end  - Html5, css, javascript, Bootstrap, Jquery
+ Back-end  - Nodejs, express-nodejs, mysql, npm

## 나의 작업

+ Nodejs 백엔드 서버 관리
+ MySQL DB 연동 및 관리
+ Admin 페이지 레이아웃 구현
+ Admin 로그인 기능 구현
+ Admin / guest 게시판 구현
